The riverscapes stream names report is intended to provide an accessible introduction to the the named streams within a specific area of interest. It pulls together data from a variety of public sources and combines them with the results from several models. The goal is to encourage exploration of the riverscape and the landscape context in which it resides.

Stream Names

Name diversity

When setlers were naming streams, lives were much more local than they are today. There was no national registry like the Geographic Names Information System. As a result, many different stream share names. This word cloud illustrates this by making the size of the name proportional to the number of individual level paths that use that name within the area of interest. Only the 200 named streams (by distinct level paths) are shown; the full list can be found in the report data folder. Note - the word cloud algorithm adjusts word sizes to fit the shape and this effect is more pronounced for small areas with just a few stream names. Consult the included data files for accurate values.

Riverscape Length

The stream names come from NHD data aggregated as part of the Riverscapes Context project. In this word cloud, the size of the name is proportional to Riverscape length and the colour represents the maximum stream order for this stream within the area of interest (darker colour = larger stream order). Only top 200 named streams (by Riverscape length) are shown; the full list can be found in the report data folder.

Riverscape Area per length

The stream names come from NHD data aggregated as part of the Riverscapes Context project. In this word cloud, the size of the name is proportional to Riverscape area divided by riverscape length and the colour represents the maximum stream order for this stream within the area of interest (darker colour = larger stream order). Only top 200 named streams (by this metric) are shown; the full list can be found in the report data folder.
